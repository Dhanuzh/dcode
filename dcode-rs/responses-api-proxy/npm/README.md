# @openai/dcode-responses-api-proxy

<p align="center"><code>npm i -g @openai/dcode-responses-api-proxy</code> to install <code>dcode-responses-api-proxy</code></p>

This package distributes the prebuilt [Dcode Responses API proxy binary](https://github.com/openai/codex/tree/main/dcode-rs/responses-api-proxy) for macOS, Linux, and Windows.

To see available options, run:

```
node ./bin/dcode-responses-api-proxy.js --help
```

Refer to [`dcode-rs/responses-api-proxy/README.md`](https://github.com/openai/codex/blob/main/dcode-rs/responses-api-proxy/README.md) for detailed documentation.
